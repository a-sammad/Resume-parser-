# Resume-parser-

Resume Parser using Python, Flask & NLP

I'm excited to share one of my recent projects—a **Resume Parser** built using **Python, Flask, SpaCy, Regular Expressions (Regex), PyPDF2, and python-docx**.

This application automatically extracts important information from resumes, making the recruitment process faster and more efficient.

 **Key Features:**
✅ Extracts Candidate Name
✅ Detects Email Address & Phone Number
✅ Identifies Skills from a custom skills database
✅ Recognizes Highest Education Level
✅ Extracts Experience (Years, Months & Internship)
✅ Supports **PDF, DOCX, and TXT** resume formats
✅ Simple and user-friendly Flask web interface

 **Technologies Used:**

* Python
* Flask
* SpaCy (NLP)
* Regular Expressions (Regex)
* PyPDF2
* python-docx
* HTML/CSS

This project helped me strengthen my understanding of **Natural Language Processing (NLP), text extraction, regex pattern matching, and Flask web application development** while building a practical solution for resume screening.
